01 / Computer Vision
Skin Cancer Detection

A CNN-based classifier that flags suspicious skin lesions from dermoscopic images, wrapped in a Flask web app for real-time predictions.

PythonTensorFlowKeras CNNOpenCVFlask
Dataset→ Preprocessing→ CNN→ Prediction→ Web App

Overview

Trains a convolutional neural network on labelled dermoscopic images to distinguish benign from malignant lesions, then serves predictions through a lightweight web interface.

Features

  • Image upload with instant inference
  • Confidence score per prediction
  • Preprocessing pipeline for contrast & noise correction

Architecture

Custom CNN with convolution, pooling and dropout layers, trained on augmented image data to reduce overfitting; OpenCV handles image normalization before inference.

Challenges & Solutions

Class imbalance in the dataset was addressed with targeted augmentation and class-weighted loss, improving recall on minority (malignant) samples.

02 / Generative AI
AI Document Agent (RAG)

A retrieval-augmented agent that answers questions grounded in a user's own PDFs, running locally via Docker with a Streamlit front end.

LangChainOpenAI APIGroq API FAISSDockerStreamlit
PDF Upload→ Chunking→ Embedding→ Vector DB→ LLM→ Answer

Overview

Uploaded documents are chunked, embedded, and stored in a FAISS vector index; user queries retrieve the most relevant chunks before being passed to an LLM for a grounded answer.

Features

  • Multi-document ingestion
  • Source-cited answers
  • Swappable LLM backend (OpenAI or Groq)

Architecture

LangChain orchestrates the retrieval chain; FAISS handles similarity search over chunk embeddings; the whole stack is containerized for reproducible deployment.

Challenges & Solutions

Long documents produced noisy retrieval — solved by tuning chunk size and overlap, and re-ranking retrieved chunks before passing them to the LLM.

Future Improvements

Add conversational memory across sessions and support for image-based (scanned) PDFs via OCR.

03 / Time Series
Bitcoin Price Prediction

Comparative forecasting of BTC price movement using classical and deep learning models on historical time-series data.

Random ForestLSTMTensorFlow Scikit-learnPandas

Overview

Historical BTC price data is engineered into lagged features and rolling statistics, then modelled with both a Random Forest baseline and an LSTM network.

Features

  • Rolling-window feature engineering
  • Side-by-side model comparison
  • Interactive prediction chart

Challenges & Solutions

Crypto price series are highly volatile and non-stationary — differencing and normalization stabilized training, and an LSTM captured longer-range dependencies better than the tree-based baseline.

Future Improvements

Incorporate on-chain and sentiment data as additional features to improve directional accuracy.
